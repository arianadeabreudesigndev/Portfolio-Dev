Tabela Periódica

Tabela periódica interativa com informações detalhadas de cada elemento.

Frontend HTML/CSS JavaScript
Este projeto foi um dos primeiros que desenvolvi na faculdade e marcou meu aprendizado em HTML e CSS Grid. Criei uma tabela periódica interativa onde cada elemento funciona como um botão que, ao ser clicado, revela uma janela com a sigla, o nome, uma breve descrição e quatro imagens que mostram onde o elemento é encontrado ou como é utilizado no dia a dia. Foi um exercício divertido e desafiador que me ajudou a entender melhor a organização de layouts e a tornar o conteúdo mais interessante e acessível.
